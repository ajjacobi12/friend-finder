What started off as a fun sideproject has (unsurprisingly) turned into something much bigger than I originally anticipated. 
At the end of the day, this is a great exercise to learn more about app development from start to finish, the logic behind it all, and gain more coding experience. 
Probably won't go anywhere, but it's been fun so far.

Intended to be a peer-to-peer location tracker with messaging that can be utilized in GPS denied environments by using mesh-networking.
